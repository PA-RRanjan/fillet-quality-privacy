# Fillet Quality App — Privacy Policy

This repository hosts the official Privacy Policy for **Fillet Quality App**, a smartphone application developed by [The Conservation Fund Freshwater Institute](https://www.conservationfund.org/freshwater-institute) for fillet color scoring and defect detection (gaping, melanosis) in aquaculture quality inspection.

The policy is published via GitHub Pages and is intended for submission to the Apple App Store and Google Play Store, as well as for general public reference.

## Key Points Covered in the Policy

- The App runs entirely on-device (smartphone); no data is transmitted to remote servers.
- Camera permission is required for fillet image capture only, and is not used for any other purpose.
- No personal information is collected (no accounts, location, analytics, or tracking).
- Inspection images/results are shared externally **only** if the user actively chooses to do so (e.g., via email or another app).
